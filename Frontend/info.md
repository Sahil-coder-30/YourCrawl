# YourCrawl - Frontend Info & Status

## Overview
The Frontend is a React application built with Vite. It serves as the user interface for the Dark Pattern Auditor platform. The application is styled using Tailwind CSS and provides various views to display the dark pattern audit results, configuration, and compliance details.

## Current State & Features
- **Routing & Navigation**: Uses `react-router-dom` for client-side routing. Key pages include:
  - `/` (Landing Page)
  - `/dashboard` (Main Dashboard)
  - `/config` (Configuration settings)
  - `/audits` (Historical or active audits)
  - `/analysis` (Deep dive into scan results)
  - `/roadmap` (Remediation roadmap for detected patterns)
  - `/compliance` (Regulatory compliance assessment)
- **UI Components**:
  - Reusable UI elements located in `src/components/ui/` (buttons, checkboxes, inputs, selects, tabs).
  - Common layout components like `Sidebar`, `TopNav`, `AppLayout`, and `StatCard`.
- **Styling**: Tailwind CSS is used extensively for utility-first styling (`@tailwindcss/postcss`).
- **Icons & Notifications**: `lucide-react` is used for icons, and `sonner` is used for toast notifications.
- **Data Management**: Currently relies on `mockData.js` for some UI testing, but connects to the Backend to display actual dark pattern scan reports.

## Key Directories
- `src/pages/`: Contains the main route components (Landing, Dashboard, Config, etc.).
- `src/components/`: Contains shared and layout components.
- `src/styles/`: Contains global CSS variables (`variables.css`).
- `src/data/`: Mock data for development.

## Next Steps / Status
The frontend provides a complete shell to visualize the markdown reports and scan results generated by the backend. Future integrations might involve streaming scan statuses or adding more interactive visualizations of the UI bounding boxes.
